To run this app:

- Make sure you download the latest version of Android Studio, available here: 
- Open Android Studio
- Go to File -> New -> Project from Version Control...
- Select Github and enter your Github credentials if you have not already done so
- Select the "diogoviveiros/MobAIlFinal" project
- Enable Developer Mode for your Android phone (preferrably running Android 11 or higher, each phone differs on how to enable developer mode, so please search how to do it on your specific device)
- Plug in your device to your computer via USB
- Allow USB debugging
- In the top bar in Android Studio, make sure your device is selected as the runnable device, then press the "Run" Button on the right of that tab.
- Wait for the Gradle to compile, and for the app to be installed on your device!
